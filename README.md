# E2E Test Automation - Playwright

This repository contains automated end-to-end tests built with Playwright.

## Application Under Test
The tests are executed using the demo website:

https://www.saucedemo.com

SauceDemo is a sample e-commerce application commonly used for test automation practice.

## Tools
- Playwright
- JavaScript
- Node.js

## Test Scenarios

Authentication
- Login with valid credentials
- Login with invalid password
